# AMY release v1.11.4

[This release](https://github.com/swcarpentry/amy/milestone/45) is intended
to comply with GDPR regulations in European Union and focuses mostly on
AMY compliance with this law.

## Bug fixes

* [Piotr][] fixed introduced in v1.11.3 bug that caused `SWC` tag to disappear
  from production server. The bug was related to changes in merging object
  functionality.

## New features

* [Jonah][] updated privacy policy location,
* [Piotr][] added option for users to publish their profiles on The Carpentries
  websites,
* [Piotr][] added a popup for users to agree to the privacy policy.

[Piotr]: https://github.com/pbanaszkiewicz
[Jonah]: https://github.com/jduckles
